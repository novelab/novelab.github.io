REMOTE CONTROL OPERATION

A project led by the social media and digital agency Uzful. Through a web interface, the user could give commands via a Sony Xperia Z3 Tablet to a robot that played a PS4 game Little Big Planet 3 in remote play.

The installation was set up at the Sony booth at the Christmas market of Champs Élysées in Paris and broadcast live on the website http://operation-remotectrl.com/ where users were able to track their own interactions with the robot in real time.

The device

Using methods and rapid prototyping tools, we were able to design a robot in very short time. The device was designed and produced locally:
- Design and production monitoring by Uzful.
- Design of the robot, electronic and Arduino programming by Novelab with the help of Beatrice Lartigue for the overall design.
- Manufacture of the robot using the digital workshop (laser cutter, 3D printer, digital milling machine ...) of ICI Montreuil by Pierrick Verger.
